I built this project to explore the power of recursive backtracking in solving constraint-satisfaction problems. The goal was to create a script that generates a fully valid 9x9 Sudoku board where every row, column, and 3x3 subgrid contains the digits one through nine without any repeats. Because a Sudoku board has millions of possible configurations, I couldn't just use random guessing; I had to implement a systematic way for the computer to "try" a number and, if it hits a dead end, backtrack to a previous step to try a different path.

The technical core of this project is the is_valid function, which serves as the "referee" for the board. Before placing any number, the program checks three distinct constraints: it scans the horizontal row, the vertical column, and the local 3x3 square. To handle the 3x3 box check, I used integer division to calculate the starting coordinates of each subgrid. This ensures that the board remains mathematically sound at every step. By combining these rules with a randomized list of numbers, the generator produces a unique board every time the script is run.

The most challenging part of this project was managing the recursion within the fill_board function. The program moves cell by cell, and if it reaches a point where no numbers from 1 to 9 can legally fit, it "un-fills" the current cell (resetting it to zero) and returns to the previous function call. This "backtracking" continues until a valid path is found. I also put effort into the visual presentation, writing a formatting loop that adds pipes and dashes so the terminal output looks like a real Sudoku paper grid rather than just a wall of numbers.

In the future, I plan to turn this generator into a playable game by adding a "difficulty" function that strategically removes numbers from the completed board while ensuring it still has a unique solution. I also want to build a Graphical User Interface (GUI) using Pygame so users can click and type numbers directly into the grid. This project was a great way to bridge the gap between simple loops and the complex recursive logic used in professional software and game AI.
